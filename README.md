# 📊 Sales & Revenue Analysis Dashboard

An interactive Sales & Revenue Analysis Dashboard developed using Python, Pandas, Plotly, and Streamlit.  
This project analyzes sales performance, revenue trends, profits, customer behavior, and business KPIs using the Superstore dataset.

---

## 🚀 Features

- 📈 Sales Trend Analysis
- 💰 Revenue & Profit Tracking
- 📊 KPI Dashboard
- 🏆 Top Performing Products
- 🌍 Region Wise Sales Analysis
- 👥 Customer Segment Analysis
- 🚚 Shipping & Delivery Insights
- 📉 Discount vs Profit Analysis
- 🔍 Interactive Data Visualizations
- 📦 Streamlit Dashboard Integration

---
